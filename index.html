Sobre os desafios

Projeto desenvolvido com o objetivo de treinar as minhas habilidades atraves de desafios que foram propostos pelo Iuri Silva (@iuricode) no servidor do Discord da Codelândia.

No servidor do Discord terá a seção "Desafios" onde vamos ter acesso a cada desafio, aos arquivos que foram utilizados (fontes, imagens, dicas...), ao link para o Figma com o design, uma sala para tirar as dúvidas e outra sala para apresentar os resultados.

Os desafios são para nível júnior, mas com o tempo eles vão se aprofundando.
